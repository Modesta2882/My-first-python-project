# My-first-python-project
This project focuses on the end-to-end process of transforming a "real-world" messy dataset into a clear, actionable narrative. Using Python, I performed extensive data cleaning and exploratory data analysis (EDA) on a complex crime dataset to identify patterns, seasonal trends, and high-risk areas.
Key Objectives
Data Wrangling: Tackled common real-world issues such as missing values, inconsistent formatting, and duplicate entries using the pandas library.

Temporal Analysis: Standardized date and time formats to analyze how crime rates fluctuate across different months, days of the week, and hours of the day.

Geospatial & Categorical Insights: Cleaned location descriptions and crime categories to visualize which areas and types of incidents were most prevalent.

Data Visualization: Created intuitive charts and graphs using Matplotlib and Seaborn to translate raw numbers into visual stories.

Technical Stack
Language: Python

Libraries: Pandas (Data Manipulation), NumPy (Numerical Analysis), Matplotlib/Seaborn (Visualization)

Environment: Jupyter Notebook

The "Why": Raw data is rarely ready for analysis. This project demonstrates my ability to bridge the gap between raw, disorganized information and data-driven storytelling, ensuring that the final insights are built on a foundation of clean, reliable data.
